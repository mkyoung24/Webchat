# 채팅방

## :calendar: 진행 기간 
- 2022년 2월 12일 ~ 2022년 3월 5일

## :page_facing_up: 프로젝트 목적
- NoSQL 데이터베이스 경험 쌓기
- 채팅 서비스를 만들어 보고 싶기 위

- WebSocket을 활용한 채팅 웹 서비스
